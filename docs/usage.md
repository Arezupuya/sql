# Usage

Run from project root:

## Init database
python3 src/cli.py init

## Seed sample data
python3 src/cli.py seed

## List tables
python3 src/cli.py list

## JSON report
python3 src/cli.py report

## Self-test
python3 src/self_test.py

## Custom DB path
python3 src/cli.py --db data/mydb.db init
